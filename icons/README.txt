**Como usar no PWA**
- No `manifest.json`, troque `name` e `short_name` para "Momozi Finance".
- Substitua as entradas de `icons` pelas novas imagens em `/icons`.
- Em `index.html`, altere o `<title>` para "Momozi Finance".
- Opcional: ajuste sua UI para usar Indigo (#4F46E5) como `--primary` e Violet/Cyan como destaques.

**Dica iOS (PWA)**
- A `apple-touch-icon.png` já está pronta.
